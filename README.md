# aws-3tier-ecs-terraform
Terraform + ECS/Fargate + RDS + ALB + CloudFront Infra
FastAPI Task Management API

A lightweight Task Management REST API built with FastAPI, SQLAlchemy, and SQLite, featuring full CRUD operations, automated testing with pytest, linting with ruff, formatting with black, and CI via GitHub Actions.

This project demonstrates backend API development best practices including:

RESTful design

Input validation with Pydantic

Database persistence with SQLAlchemy

Dependency injection

Automated testing

CI/CD pipeline integration

Clean project structure
